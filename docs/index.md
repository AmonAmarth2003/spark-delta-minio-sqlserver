# spark-delta-minio-sqlserver

Projeto acadêmico demonstrando um pipeline de dados completo utilizando
Apache Spark, Delta Lake, MinIO e SQL Server.

O foco do projeto é apresentar:
- Extração de dados de banco relacional
- Persistência em Object Storage (MinIO)
- Conversão para Delta Lake
- Operações DML (INSERT, UPDATE, DELETE)
- Histórico e Time Travel com Delta Lake
